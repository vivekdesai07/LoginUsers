== README

Ruby on Rails micro project to test authentication from scratch by hand.
This project is based on this simple tutorial:

http://www.sitepoint.com/rails-userpassword-authentication-from-scratch-part-i/

All forms, labels, inputs and submit text are in Portuguese-PT

General Info:

Ruby version: 2.1.1

Rails version: 4.1.1

System dependencies: see gem file

Database: POSTGRESQL

<tt>rake doc:app</tt>.
